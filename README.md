# Brookode Website

Brookode is a modern business website built for showcasing web development services, portfolio work, and contact options for client enquiries.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
npm run preview
```
